## Dependencies

1. Rust toolchain (stable, 2021 edition)
2. `num-bigint = "0.4"` (listed in Cargo.toml)

## Build and Test

- Build: `cargo build`
- Run all solutions: `cargo run`
- Run unit tests: `cargo test`
- Run single test: `cargo test euler_001`
